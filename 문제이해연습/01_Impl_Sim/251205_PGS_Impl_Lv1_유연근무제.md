**② Python Code Snippet (패턴 저장)**
> 나중에 써먹을 수 있는 핵심 코드 조각
- **시간 연산** : HHMM 형태의 정수를 다룰 때, 단순 덧셈이 아닌 분(Minute) 단위의 올림 처리가 필수적
```python
def add_time(time_val, minutes):
    h, m = divmod(time_val, 100)
    m += minutes
    if m >= 60:
        h += 1
        m -= 60
    return h * 100 + m
```
